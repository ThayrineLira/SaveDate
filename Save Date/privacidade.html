Privacidade

Dados claros, controle simples e nenhuma surpresa.

Esta pagina explica como o Save Date usa informacoes no prototipo atual e quais cuidados entram em uma versao pronta para usuarios reais.

Transparencia

Mostramos quais dados sao usados para login, favoritos, filtros e suporte.

Controle

As preferencias ficam no navegador e podem ser apagadas pelas configuracoes do app ou do browser.

Minimizacao

O app deve pedir apenas o necessario para recomendar lugares e manter a conta funcionando.

Dados que o prototipo pode guardar

Na versao atual, o Save Date usa armazenamento local do navegador para simular funcionalidades de produto. Isso pode incluir:

  • nome, e-mail, tipo de conta e estado de login;
  • lugares salvos, avaliacoes, comentarios e preferencias;
  • dados de cadastro de estabelecimento preenchidos pelo usuario;
  • chamados enviados pela pagina de suporte.

Localizacao e mapas

Quando voce usa recursos como "perto de mim", o navegador pode pedir permissao para acessar sua localizacao. O Save Date usa essa informacao para ordenar lugares por distancia. Se voce negar a permissao, a busca continua funcionando com filtros manuais.

Algumas paginas usam servicos externos, como ViaCEP, OpenStreetMap, Leaflet, Google Maps e fontes hospedadas externamente. Esses servicos podem receber dados tecnicos basicos da navegacao, como IP e informacoes do navegador.

Como seus dados sao protegidos

Como este projeto ainda e estatico e academico, dados sensiveis nao devem ser usados em ambiente publico. Em uma versao de producao, senhas seriam protegidas por backend seguro, criptografia adequada, controle de sessao e politicas de acesso.

Nao use senhas reais neste prototipo. Ele simula login no navegador para demonstrar o fluxo do produto.

Como apagar informacoes

Voce pode limpar dados pelo proprio navegador apagando o armazenamento local do site. Quando disponivel, use tambem a tela de configuracoes do Save Date para remover preferencias e dados salvos.

Duvidas sobre dados ou remocao podem ser enviadas pela pagina de suporte e contato.
